MINHAS ESPECIALIDADES.

html

HTML é como um LEGO para construir páginas da web. Com ele, você usa tags (pequenos comandos) para definir a estrutura e o conteúdo de um site.

css

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é a linguagem responsável por definir a aparência visual de páginas web construídas com HTML. Enquanto o HTML estrutura o conteúdo, o CSS molda a apresentação, controlando cores, fontes, tamanhos, espaçamentos e layout.

javascript

JavaScript é a linguagem de programação que traz vida às páginas web. Enquanto o HTML estrutura o conteúdo e o CSS define a aparência, o JavaScript adiciona interatividade e dinamismo. Com ele, você pode criar elementos que respondem a ações do usuário, como botões que executam funções, formulários que validam dados, animações e muito mais.
